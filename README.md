# Portfolio

## Technologies Used
Frontend:
Next.js / React

## System Requirements
Make sure you have Node.js installed on your system.


## Setup

### Frontend

1. Clone the repository.
2. In the project folder, navigate to the frontend directory.

        cd frontEnd

3. Install dependencies.

        npm install

4. Start the Next.js application.

        npm run dev
